# CNN Architecture Comparison – Dog vs Cat Classification

This project compares multiple deep learning CNN architectures for binary image classification (Dog vs Cat).

## Architectures

* ResNet50
* InceptionV3
* MobileNetV2
* EfficientNetB0
* Xception

## Workflow

EDA → Preprocessing → Transfer Learning → Training (10 epochs) → Evaluation → Comparison

## Dataset

Kaggle Dogs vs Cats dataset (not included due to size).

## Tech Stack

TensorFlow, Keras, Scikit-learn, Matplotlib, Google Colab (GPU)
